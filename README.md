# GCP SKILLS JOURNEY

This repository contains my completed hands-on labs, exercises, and challenge activities from **Google Cloud Skills Boost**, with a strong focus on **SRE (Site Reliability Engineering)**, **DevOps**, and **System Engineering** practices using Google Cloud Platform (GCP).

The content here reflects real-world cloud workloads, infrastructure tasks, automation, monitoring, and troubleshooting scenarios across various Google Cloud services.

## Purpose of This Repository

The main goals of this repo are:
- To document my learning journey in Google Cloud SRE & DevOps
- To build a strong reference guide for real-world cloud engineering tasks
- To showcase hands-on cloud skills and practical implementation experience
- To prepare for roles involving GCP DevOps/SRE/System Engineering
